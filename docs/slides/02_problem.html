Glyphs
Filled - Capability exists
Empty - Gap / not provided
Square - Different category
Key Difference
Sandboxes answer: "Can agent escape?"

Enterprises need: "Can agent call this tool with these parameters?"

These are different questions requiring different architectures.
Insight

Enterprise adoption is blocked not by technical capability, but by provability. CISOs need to prove agents can't do things. SOC2, FedRAMP, HIPAA, EU AI Act all require written policy + enforcement + audit.
